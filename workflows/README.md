# Workflows

This directory is reserved for Argo WorkflowTemplates during later milestones.

For `Q2`, the required deliverables are satisfied by:

- `tf/kvm/` for provisioning
- `ansible/` for cluster bootstrap and deployment
- `k8s/` for canonical Helm charts

When the project reaches full integration, this directory should hold the ChefMate equivalents of:

- training workflow templates
- deployment/promotion workflow templates
- scheduled retraining workflows
